**Endpoint 1 — POST /api/presentations/transcribe (D-05):**
```
Zod body schema: { audioData: z.string().min(1) }
1. safeParse body → 400 on failure
2. Resolve Groq key: getRuntimeGroqKey() || (await storage.getChatIntegration("groq"))?.apiKey
3. 503 if no key: "Groq API not configured — set GROQ_API_KEY or configure in admin"
4. Strip data URL prefix: base64Data = audioData.replace(/^data:audio\/\w+;base64,/, "")
5. Convert: buffer = Buffer.from(base64Data, "base64")
6. Dynamic import Groq SDK: const Groq = (await import("groq-sdk")).default
7. groq.audio.transcriptions.create({ file: new File([buffer], `prompt_${Date.now()}.webm`, { type: "audio/webm" }), model: "whisper-large-v3", response_format: "text" })
8. Cast result: const text = (transcription as unknown as string).trim()
9. res.json({ transcription: text })
```

**Endpoint 2 — POST /api/presentations/generate (D-08 through D-14):**
```
Zod body schema: { title: z.string().min(1).max(200), prompt: z.string().min(1).max(8000) }
1. safeParse body → 400 on failure
2. Resolve Gemini key: getRuntimeGeminiKey() || process.env.GEMINI_API_KEY || (await storage.getChatIntegration("gemini"))?.apiKey
3. 503 if no key: "Gemini API not configured — set GEMINI_API_KEY or configure in admin"
4. const model = process.env.GEMINI_PRESENTATION_MODEL || "gemini-2.0-flash"
5. const guidelinesRow = await storage.getBrandGuidelines()
6. const guidelines = guidelinesRow?.content ?? ""
7. const client = getGeminiClient(apiKey)
8. response = await client.chat.completions.create({ model, max_tokens: 8192, messages: [{ role: "system", content: buildGeneratorSystemPrompt(guidelines) }, { role: "user", content: parsed.data.prompt }], tools: [GENERATE_SLIDES_TOOL], tool_choice: { type: "function", function: { name: "generate_slides" } } })
9. const toolCall = response.choices[0]?.message?.tool_calls?.[0]
10. If !toolCall || toolCall.type !== "function": return res.status(500).json({ message: "Gemini did not invoke the generate_slides tool" })
11. const rawInput = JSON.parse(toolCall.function.arguments)  ← arguments is a JSON string, must parse
12. const validation = z.array(slideBlockSchema).safeParse(rawInput.slides)
13. If !validation.success: return res.status(422).json({ message: "Generated slides failed validation", errors: validation.error.errors })
14. const slug = await buildUniquePresentationSlug(parsed.data.title)
15. const presentation = await storage.createPresentation({ title: parsed.data.title, slug, slides: validation.data, guidelinesSnapshot: guidelines })
    ← guidelinesSnapshot passes DIRECTLY to storage (NOT through insertPresentationSchema Zod, which omits it)
16. res.status(201).json({ id: presentation.id, slug: presentation.slug })
```
