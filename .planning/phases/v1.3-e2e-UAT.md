## Issue Log

_Bugs found and fixed during UAT:_
- Duplicate "Avatar" / "Background Image" labels on uploaders
- Live Preview iframe clipped by aspect-ratio constraint — switched to h-full fill
- Delete button missing for uploaded assets — added hover trash overlay + DELETE endpoint
- Primary color not applied on public page — Tailwind opacity modifier broken with CSS vars; fixed with inline style + JS hover handlers
- Social Links redesigned: platform selector + username-only field
- `h1` title barely visible — added explicit `text-white`
- White background artifact on rounded corners — Links page loaded without `dark` class; forced dark mode on mount + replaced `<Card>` with plain div
