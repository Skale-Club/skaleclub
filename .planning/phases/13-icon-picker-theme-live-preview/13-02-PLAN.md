    ```tsx
    export interface ThemeEditorProps {
      theme: LinksPageTheme;
      onChange: (patch: Partial<LinksPageTheme>) => void;
    }
    ```

    **Implementation:**

    1. **Local state mirrors the props** so color/text typing is smooth; debounced save fires upstream:
       ```tsx
       const [primary, setPrimary] = useState<string>(theme.primaryColor ?? DEFAULT_LINKS_PAGE_THEME.primaryColor);
       const [bg, setBg] = useState<string>(theme.backgroundColor ?? DEFAULT_LINKS_PAGE_THEME.backgroundColor);
       const [gradient, setGradient] = useState<string>(theme.backgroundGradient ?? '');
       ```

       Re-sync when the upstream theme changes (after save completes and refetches):
       ```tsx
       useEffect(() => {
         setPrimary(theme.primaryColor ?? DEFAULT_LINKS_PAGE_THEME.primaryColor);
         setBg(theme.backgroundColor ?? DEFAULT_LINKS_PAGE_THEME.backgroundColor);
         setGradient(theme.backgroundGradient ?? '');
       }, [theme.primaryColor, theme.backgroundColor, theme.backgroundGradient]);
       ```

    2. **Debounced save** (400ms) per field. Use a single scheduler pattern — each field change sets local state then schedules an upstream patch:
       ```tsx
       const saveTimer = useRef<ReturnType<typeof setTimeout> | null>(null);
       const schedule = (patch: Partial<LinksPageTheme>) => {
         if (saveTimer.current) clearTimeout(saveTimer.current);
         saveTimer.current = setTimeout(() => onChange(patch), 400);
       };
       ```

       Hex validation: only `schedule` when the value matches `/^#[0-9a-fA-F]{6}$/` (matches the schema regex). Invalid intermediate typing does not fire upstream.

       Gradient: no validation — free-form string (intentionally permissive; server Zod accepts any string).

    3. **Render** (one AdminCard, FormGrid cols={1}):
       ```tsx

             </div>
           </div>
           <FormGrid cols={1}>
             {/* Primary color */}
             <div className="space-y-2">
               <Label htmlFor="primaryColor">{t('Primary Color')}</Label>
               <div className="flex items-center gap-2">
                 <input

                   value={primary}

                   onChange={(e) => {
       ```
