    1. **Debounced search state**: `const [query, setQuery] = useState('')` + `const [debouncedQuery, setDebouncedQuery] = useState('')`. In a `useEffect(() => { const h = setTimeout(() => setDebouncedQuery(query), 250); return () => clearTimeout(h); }, [query])`. Needs `useEffect` import too.

    2. **Lucide icon list** (memoized, filtered):
       ```tsx
       const lucideEntries = useMemo(() => {
         // LucideIcons is a giant namespace — filter to PascalCase-named components only.
         return Object.entries(LucideIcons)
           .filter(([name, v]) =>
             /^[A-Z][a-zA-Z0-9]+$/.test(name) &&
             typeof v === 'object' && // lucide-react icons are forwardRef exotic components (type 'object')
             name !== 'Icon' &&
             name !== 'LucideIcon' &&
             !name.endsWith('Icon') // lucide re-exports `FooIcon` aliases; keep the canonical `Foo`
           );
       }, []);

       const filtered = useMemo(() => {
         const q = debouncedQuery.trim().toLowerCase();
         if (!q) return lucideEntries.slice(0, 200); // cap idle list to 200
         return lucideEntries
           .filter(([name]) => name.toLowerCase().includes(q))
           .slice(0, 200);
       }, [debouncedQuery, lucideEntries]);
       ```

    3. **Trigger button**: a 32×32 square rendering the currently-selected icon:
       ```tsx
       function TriggerIcon() {
         if (iconType === 'lucide' && iconValue) {
           const Cmp: any = (LucideIcons as any)[iconValue];
           if (Cmp) return <Cmp className="w-5 h-5" />;
         }
         if (iconType === 'upload' && iconValue) {
           return <img src={iconValue} alt="" className="w-5 h-5 object-contain" />;
         }
         return <LinkIcon className="w-5 h-5" />;
       }
       ```
