<objective>
## Goal
Fechar todos os gaps de cobertura de tradução: Footer exibindo `tagline` sem `t()`, 5 strings hardcoded em Links.tsx, e chaves estáticas faltando em `translations.ts`.

## Purpose
Após o Plan 01 resolver o flash com o preload, este plano garante que TODA string visível ao usuário em modo PT passa pelo sistema `t()`. Strings estáticas conhecidas vão direto ao dict; strings dinâmicas do DB atingem o batch/preload.

## Output
- `Footer.tsx`: `{tagline}` → `{t(tagline)}`
- `Links.tsx`: 5 strings hardcoded → `t()` com chaves adicionadas a `translations.ts`
- `translations.ts`: chaves novas para strings de Links.tsx
</objective>
